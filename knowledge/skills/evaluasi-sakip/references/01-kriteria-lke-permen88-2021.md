## Cara Penilaian (WAJIB dibaca)

**Penilaian per KRITERIA = INPUT dari bukti.** **Roll-up (sub-komponen/komponen/total) = DIHITUNG dengan rumus** — kolom roll-up di LKE ini TIDAK ber-rumus Excel, jadi sistem WAJIB menghitung sendiri (bukan mengetik bebas).

### Langkah per kriteria
1. Baca **data dukung** yang relevan secara menyeluruh.
2. Cocokkan kondisi nyata dengan **rubrik parameter A/B/C** kriteria tersebut.
3. Tetapkan nilai huruf **A / B / C**. *(Keterangan TIDAK diisi per kriteria — diringkas di baris sub-komponen.)*

**Konversi huruf kriteria -> angka:**

| Huruf | Makna | Angka |
|-------|-------|-------|
| **A** | Terpenuhi | **100** |
| **B** | Sebagian terpenuhi | **50** |
| **C** | Belum terpenuhi | **0** |

### Roll-up (DIHITUNG oleh sistem)

```
Persen sub-komponen   = Sigma(angka kriteria) / (jumlah kriteria x 100) x 100%
Nilai Akhir sub-komp. = (Predikat / 100) x Bobot sub-komponen
Nilai AKIP (total)    = Sigma semua Nilai Akhir sub-komponen   (maks 100)
```

Persen -> predikat sub-komponen lewat tabel skala di bawah.

### Skala Predikat Sub-Komponen

| Predikat | Angka | Kondisi Pemenuhan Kriteria |
|----------|-------|---------------------------|
| **AA** | 100 | 100% **dan** ada upaya inovatif, layak jadi percontohan nasional |
| **A** | 90 | 100% **dan** ada beberapa upaya yang bisa dihargai |
| **BB** | 80 | 100% sesuai mandat kebijakan |
| **B** | 70 | >75%-100% |
| **CC** | 60 | >50%-75% |
| **C** | 50 | >25%-50% |
| **D** | 30 | >0%-25% |
| **E** | 0 | 0% (tidak ada upaya) |

### Aturan MAKSIMAL BB (WAJIB)

Penilaian AI **maksimal BB** — **JANGAN pernah memberi A atau AA**. Predikat A/AA hanya boleh ditetapkan melalui **professional judgement APIP (manusia)** bila ada bukti inovasi/percontohan. Predikat hasil AI hanya: **BB, B, CC, C, D, E**. Bila 100% terpenuhi -> tetap **BB** (bukan A/AA).

### Kontrak Kolom LKE APIP (di mana menulis)

Identifikasi kolom lewat **label header** LKE (mis. "Predikat APIP", "Nilai APIP", "Keterangan APIP") — bukan huruf kolom mati. **JANGAN** menyentuh kolom Penilaian Mandiri (auditee) & sel formula.

**Baris KRITERIA (INPUT dari bukti):**

| Kolom (header) | Isi |
|---|---|
| Predikat APIP | huruf **A / B / C** |
| Nilai APIP | angka **100 / 50 / 0** |
| Keterangan APIP | *(dikosongkan)* |

**Baris SUB-KOMPONEN (DIHITUNG + ringkasan):**

| Kolom (header) | Isi |
|---|---|
| % APIP | Persen sub-komponen (rumus di atas) |
| Nilai APIP | nilai |
| Predikat APIP | predikat sub-komponen (**maksimal BB**) |
| Nilai Akhir APIP | (Predikat/100) x Bobot |
| % APIP (akhir) | persentase |
| Keterangan APIP | **RINGKASAN** catatan seluruh kriteria di bawahnya (mis. "Belum terdapat pedoman teknis dan dokumen perencanaan belum memenuhi standar") |
| Kolom penanda (tambahan, paling kanan) | bila predikat = **BB** -> tulis "Perlu professional judgement APIP (potensi A/AA bila ada inovasi/percontohan)"; selain BB -> **kosong** |

**Baris KOMPONEN / TOTAL (DIHITUNG):** Nilai Akhir APIP, % APIP, dan **Total Nilai AKIP** = Sigma nilai akhir sub-komponen.
