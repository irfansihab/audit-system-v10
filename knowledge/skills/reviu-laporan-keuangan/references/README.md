# References — Reviu Laporan Keuangan (LK K/L)

## C. Payung UU (konteks)

- **UU 17/2003** (Keuangan Negara) · **UU 1/2004** (Perbendaharaan Negara) · **UU 15/2004** (Pemeriksaan Pengelolaan & Tanggung Jawab Keuangan Negara).

## D. Objek & data dukung yang diperiksa (bukan kriteria)

LRA · Neraca · LO · LPE · CaLK (LAK/LPSAL untuk BUN) · kertas kerja rekonsiliasi (SAKTI↔SPAN, internal, **BMN/SIMAK-BMN**, kas) · daftar suspend/selisih · LK periode sebelumnya · LHP BPK + status TLHP.

## Catatan pengisian

- Tiap file kriteria memuat: judul + nomor + tahun · pasal/PSAP relevan (verbatim bila penting) · cara pakai sebagai kriteria di KKP.
- **Currency**: regulasi akuntansi cepat berubah — bila status ragu, tandai "perlu konfirmasi berlaku" dan jangan jadikan deviasi terkonfirmasi (doktrin `panduan-format-umum`).
